These are the steps to push a github repository

First lets discuss what is git?
git is a free and open source distributed control system designed to handle everything from small to very projects with speed and efficiency

Here are the steps to push your code to your own repository 


        1.Create a folder specifically for your projects
          #mkdir (Tabigue)

        2.git init 
        The git init command creates a new Git repository.

        3.git add README.md 

        4.git commit -m "first commit"
        For projects that already have some code files written and ready to go, the initial commit

        5.git branch -M main 
        In Git, "main" (or "master") is a naming convention for a branch

        6.git remote add origin  https://github.com/bonvonn/tabigue.git

        7.git push -u origin main 
        The conventional name for the primary version of a repository

        8.and dont forget to git status The git status command displays the state of the working directory and the staging area

        9.now your ready to push your code in your github repository
